What's Included

Everything your CarThing deserves

A complete, polished OS with features Spotify never gave you — and a desktop companion that makes setup effortless.

Spotify Controls

Play, pause, skip, and adjust volume via the touchscreen or physical dial — with a full media player showing album art, progress, shuffle, and repeat.

Customizable Shortcuts

Add up to 8 app shortcuts with Font Awesome icons or custom images. Assign your favorites to the four physical preset buttons on top.

Desktop Companion

A guided installer and companion app for Linux (Windows & Mac coming soon) that keeps your CarThing connected and up to date — no command line needed.

OTA Updates

Updates are delivered automatically via GitHub Releases. See a new version? Hit Download, then Install — done.

Audio Visualizer & Art Styles

Live audio visualizer in the media player, with multiple album art background styles including full bleed and thumbnail modes.

Lock Your Computer

Lock your PC directly from the CarThing menu — or double-tap the M button for an instant shortcut.

Sleep Timer & Screensaver

Set a sleep timer or let the built-in screensaver kick in automatically when the device is idle.

Restore to Stock

Changed your mind? Restore your CarThing to the original Spotify software at any time from the system menu.

LumiThing is in Beta

Expect occasional bugs. If you run into issues, please open a report on GitHub or join the Discord — every report helps make LumiThing better.

Getting Started

Up and running in minutes

The desktop companion handles everything — no terminal required.

  1. Download LumiThing

    Grab the latest release for your platform using the button on this page.

  2. Run the Installer

    Open the downloaded file and follow the installation prompts.

  3. Connect Your CarThing

    Open the LumiThing desktop companion and follow the guided setup to flash your CarThing.

  4. You're Done

    LumiThing is now running on your device. Future updates install automatically.
